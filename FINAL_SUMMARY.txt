## 🐛 Quick Troubleshooting

### Backend Won't Start
```batch
# Check Python
python --version

# Reinstall packages
cd backend
call venv\Scripts\pip install -r requirements.txt --force-reinstall
```

### Frontend Won't Start
```batch
# Check Node
node --version
npm --version

# Reinstall dependencies
cd frontend
npm install
```

### Database Connection Error
- Verify .env has correct Supabase URL and keys
- Check internet connectivity
- Visit https://app.supabase.com to verify project is active
